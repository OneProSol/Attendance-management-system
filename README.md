### 一、相关技术

- 前端：`Vue`、`HTML` / `CSS` / `Javascript`。
- 后端：`Springboot`等。
- 数据库：`MySQL`

### 二、开发软件（列出的软件其一均可运行）

- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`

### 三、部署所需环境

- 后端：`jdk1.8`。
- 前端：`node`、`vue-cli` 、`npm`  。
- 数据库：`mysql5.7以上`

### 四、功能描述及功能图展示

系统分为：员工、考勤员、管理员。

员工功能：1. 登录、2. 打卡、3. 加班记录、4. 请假记录、5. 出差记录、6. 考勤统计

考勤员功能：1. 登录、2. 打卡记录、3. 考勤统计

管理员功能：1. 登录、2. 用户管理、3. 加班审批、4. 请假审批、5. 出差审批、6. 考勤统计
